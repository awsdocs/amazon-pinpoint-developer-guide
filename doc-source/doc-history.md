+ **Latest documentation update:** May 27, 2022

| Change | Description | Date | 
| --- |--- |--- |
| [Journey and campaign execution metrics](#doc-history) | New analytic metrics have been added for journeys and campaigns\. | [Creating an interface VPC endpoint for Amazon Pinpoint](#doc-history) | Amazon Pinpoint now supports interface VPC endpoints\. For more information, see [Encryption in transit](https://docs.aws.amazon.com/pinpoint/latest/developerguide/security-data-protection-encryption.html#security-data-protection-encryption-transit)\. | March 20, 2023 | 
| [Amazon Pinpoint quotas](#doc-history) | Updated the process for requesting a quota increase for campaigns, journey's and In\-app messages\. For more information, see [Amazon Pinpoint quotas](https://docs.aws.amazon.com/pinpoint/latest/developerguide/quotas.html)\. | December 16, 2022 |
